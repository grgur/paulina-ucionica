1. Što je linearna jednadžba? 🤔

📘 Definicija

Linearna jednadžba je jednakost dviju strana koja sadrži jednu nepoznanicu (najčešće x).

Cilj je pronaći vrijednost x koja čini jednadžbu istinitom.

🟢 Primjer iz života

Zamislimo vagu. ⚖️

Na lijevu stranu staviš 2 kutije + 3 kg.

Na desnu stranu staviš 11 kg.

Koliko teži jedna kutija? To je jednadžba:

2x + 3 = 11

Odgovor: jedna kutija teži 4 kg. ✅

📐 Opći oblik
ax + b = 0

gdje je a ≠ 0 (koeficijent uz x) i b je slobodni član.

"Linearna" dolazi od riječi linija.

Kad nacrtaš graf ove jednadžbe, dobiješ ravnu liniju — zato se zove linearna!

Nepoznanica x nema eksponent (x², x³...) — uvijek je samo x na prvu potenciju.

2. Pravila rješavanja 🔧

🔶 Zlatno pravilo jednadžbi

Što god napraviš jednoj strani jednadžbe, moraš napraviti i drugoj!

Ako dodaš 5 na lijevu stranu → dodaj 5 i na desnu.

Ako množiš lijevu s 3 → pomnoži i desnu s 3.

💡 Trik: prebacivanje

Kad član prebaciš na drugu stranu jednadžbe, promijeni mu predznak:

+ postaje −   i   − postaje +

Ako se množi → postaje dijeljenje   i   obrnuto

Primjer korak po korak

🎯Riješi: 3x + 7 = 22
1
3x + 7 = 22
Zapišemo jednadžbu.
2
3x = 22 − 7
Prebacimo +7 na desnu stranu → postaje −7.
3
3x = 15
Izračunamo desnu stranu: 22 − 7 = 15.
4
x = 15 ÷ 3
Koeficijent 3 množi x → prebacimo ga kao dijeljenje.
✓
x = 5
Rješenje! Jedna kutija teži 5. 🎉

3. Jednadžbe sa zagradama 🔓

🔶 Prvo: otkloni zagrade!

Kad imaš zagradu, pomnoži sve unutar zagrade s brojem ispred nje.

3 · (x + 2) = 3·x + 3·2 = 3x + 6
⚠️ Pazi na minus ispred zagrade!

Minus ispred zagrade mijenja predznak svim članovima unutar zagrade:

−(2x − 5) = −2x + 5

Plus postaje minus, minus postaje plus!

Primjer korak po korak

🎯Riješi: 2(3x − 1) + 4 = 3(x + 5)
1
2(3x − 1) + 4 = 3(x + 5)
Zapišemo jednadžbu.
2
6x − 2 + 4 = 3x + 15
Množimo zagrade: 2·3x=6x, 2·(−1)=−2, 3·x=3x, 3·5=15.
3
6x + 2 = 3x + 15
Sredimo lijevu stranu: −2 + 4 = 2.
4
6x − 3x = 15 − 2
x-ove na lijevo, brojeve na desno. Predznaci se mijenjaju!
5
3x = 13
Izračunamo obje strane.
✓
x = 13/3
Rješenje! Može biti i razlomak — to je OK. 🎉

4. Jednadžbe s razlomcima 🧩

🔶 Cilj: riješiti se razlomaka!

Kad jednadžba ima razlomke, pronađi NZV (najmanji zajednički višekratnik) svih nazivnika.

Pomnoži obje strane jednadžbe s NZV-om — i razlomci nestanu! 🎉

💡 Kako naći NZV?

NZV je najmanji broj koji se dijeli sa svim nazivnicima.

Primjer: nazivnici su 2, 3 i 6 → NZV = 6

Jer se 6 dijeli s 2 ✅, s 3 ✅ i sa 6 ✅.

Primjer korak po korak

🎯 Riješi: x2 + x3 = 5
1
x/2 + x/3 = 5
Zapišemo jednadžbu. Nazivnici su 2 i 3.
2
NZV(2, 3) = 6
Tražimo NZV. 6 se dijeli i s 2 i s 3.
3
6·(x/2) + 6·(x/3) = 6·5
Množimo SVE s NZV-om = 6.
4
3x + 2x = 30
6÷2=3 pa 3·x=3x. 6÷3=2 pa 2·x=2x. Razlomci su nestali!
5
5x = 30
Zbrojimo: 3x + 2x = 5x.
✓
x = 6
30 ÷ 5 = 6. Gotovo! 🎉

5. Uvjeti jednadžbe ⚡

Ovo je jako važan dio! Objašnjavamo zašto i kako se postavljaju uvjeti.

🚫 Zabranjeno dijeljenje!

Dijeljenje nulom je zabranjeno! 🚫

Nula ne može biti u nazivniku razlomka.

Zato kad imaš x u nazivniku, moraš napisati uvjet.

Zašto?

Koliko je 10 ÷ 0 = ?

🤯 NE POSTOJI!

Ne postoji broj koji pomnožen s 0 daje 10.

Zato: nazivnik ≠ 0. Uvijek!

📘 Kako odrediti uvjete?

3 koraka za uvjete:

1
Pogledaj SVE nazivnike u jednadžbi.
2
Pronađi one koji sadrže x (nepoznanicu).
3
Izjednači svaki takav nazivnik s 0 i riješi → te vrijednosti x-a su ZABRANJENE.

Primjer 1 — Jednostavno

🎯 Jednadžba: 3x = 6
👀
Nazivnik = x
Gledamo: koji nazivnik sadrži x? Ovdje je x SAM u nazivniku.
🚫
x ≠ 0
x ne smije biti 0 jer bi nazivnik bio 0 → dijeljenje nulom!
✓
Uvjet: x ≠ 0
Pišemo uvjet PRIJE nego počnemo rješavati. ✍️

Primjer 2 — Izraz u nazivniku

🎯 Jednadžba: 5x − 3 = 2
👀
Nazivnik = (x − 3)
Cijeli izraz (x − 3) je u nazivniku.
🚫
x − 3 ≠ 0
Nazivnik ne smije biti 0. Postavljamo jednadžbu: (x − 3) ≠ 0.
🔧
x ≠ 0 + 3
Rješavamo kao malu jednadžbu: prebacimo −3 na drugu stranu.
✓
Uvjet: x ≠ 3
Ako bi x bio 3, nazivnik bi bio 3−3=0. A to je zabranjeno! 🚫

Primjer 3 — Više nazivnika

🎯 Jednadžba: 2x + 1x − 5 = 4
👀
Nazivnici: x i (x − 5)
Imamo DVA nazivnika s x-om. Trebamo dva uvjeta!
🚫
1. x ≠ 0
Prvi nazivnik je x → x ne smije biti 0.
🚫
2. x − 5 ≠ 0 → x ≠ 5
Drugi nazivnik je (x − 5) → x ne smije biti 5.
✓
Uvjeti: x ≠ 0, x ≠ 5
Oba uvjeta moraju biti ispunjena! Pišemo ih prije rješavanja.

Primjer 4 — Složeni izraz u nazivniku

🎯 Jednadžba: 72x + 4 = 1
👀
Nazivnik = (2x + 4)
Izraz s x-om u nazivniku.
🚫
2x + 4 ≠ 0
Izraz u nazivniku ne smije biti 0.
🔧
2x ≠ −4
Prebacimo +4 na drugu stranu → −4.
✓
Uvjet: x ≠ −2
−4 ÷ 2 = −2. Provjera: 2·(−2) + 4 = 0 → zabranjeno!
💡 Zapamti recept!

1. Pronađi sve nazivnike koji sadrže x.

2. Za svaki: stavi nazivnik ≠ 0.

3. Riješi tu malu nejednadžbu → dobiješ zabranjenu vrijednost.

4. Napiši: x ≠ (zabranjena vrijednost).

5. Tek onda počni rješavati jednadžbu!

6. Provjera rješenja ✅

🔶 Uvijek provjeri!

Nakon rješavanja, uvrsti x natrag u originalnu jednadžbu.

Ako lijeva strana = desna strana → rješenje je točno! ✅

Primjer provjere

🟢Jednadžba: 2x + 3 = 11, dobili smo x = 4
1
L = 2 · 4 + 3 = 8 + 3 = 11
2
D = 11
✓
L = D = 11 ✅ Rješenje je točno!
⚠️ Pazi kod jednadžbi s uvjetima!

Ako rješenje krši uvjet → jednadžba nema rješenje!

Primjer: ako uvjet kaže x ≠ 3, a ti dobiješ x = 3 → rješenje ne postoji.
